# Tutorial 02 — Run the First AI Cycle

## Goal

Execute one controlled AI task.

## Before starting

Confirm that `04_NEXT_TASK.md` contains one task only.

## AI request

```txt
Read docs/START_HERE.md and execute docs/04_NEXT_TASK.md.
Follow AISDD response format.
```

## After the AI responds

Check:

- Did it change only required files?
- Did it update documentation?
- Did it list facts, assumptions, unknowns, and risks?
- Did it update handoff?
